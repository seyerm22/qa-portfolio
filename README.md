# Michelle Reyes QA Portfolio

Learning software testing through hands-on QA projects, including manual testing, test cases, bug reports, and automation practice.

## Projects
### SauceDemo Website Testing
Website tested: https://www.saucedemo.com
Purpose: Practice manual QA testing of login functionality and basic shopping cart behavior.

Test Cases
TC01 – Login with valid credentials
Expected result: User successfully logs into the inventory page.

TC02 – Login with incorrect password
Expected result: Error message appears and user is not logged in.

TC03 – Login with empty username and password
Expected result: Validation error message appears and login is blocked.
